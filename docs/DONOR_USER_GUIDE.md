# Donor User Guide

## Account Setup

1. Register as donor.
2. Provide blood type, city, and contact details.
3. Accept privacy consent.

## Donor Dashboard

Main sections:

- Profile update
- Availability toggle
- Incoming blood requests
- Donation history

## Responding to Requests

1. Review incoming request details.
2. Choose `Accept` or `Decline`.
3. Track your response state from dashboard.

## Eligibility Rules

- Minimum donation interval is 56 days.
- Availability must be enabled to receive matching requests.

## Best Practices

- Keep blood type and city updated.
- Set availability accurately.
- Respond quickly to emergency requests.
