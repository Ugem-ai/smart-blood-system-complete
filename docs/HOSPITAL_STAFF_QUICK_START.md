# Hospital Staff Quick Start Guide

**Smart Blood System for Emergency Blood Donation Coordination**

---

## 🚀 THE 60-SECOND VERSION

**You have an emergency blood request? Here's what to do:**

### Step 1: Login (30 seconds)
```
URL: prc-blood.org/hospital/dashboard
Email: [your hospital email]
Password: [your password]
```

### Step 2: Create Request (2 minutes)
1. Click **"Create Request"** module
2. Fill in:
   - Blood Type (B+, O-, etc.)
   - Units Needed (1-20)
   - Urgency: **Critical** if emergency
   - Location: Hospital address
   - Date needed: Today or tomorrow
3. Click **Submit**

### Step 3: View Matched Donors (10 seconds)
✅ **System automatically finds top 10 donors**
- See ranked list by compatibility score
- Rank #1 = Best match (closest + most reliable)
- Yellow highlight = Top 3 (contact these first)

### Step 4: Notify Donors (5 seconds per donor)
1. Click **"Notify"** on Rank #1 donor
2. Donor gets SMS + in-app notification **instantly**
3. Repeat for Rank #2 and #3 if needed

### Step 5: Track Responses (Real-time)
1. Go to **"Response Tracker"** module
2. Select your request
3. Watch live updates:
   - ✅ Green = Accepted (ready to donate!)
   - ❌ Red = Declined
   - ⏳ Yellow = Still thinking

### Step 6: Confirm Donor
1. When donor accepts (green), click **"Confirm"**
2. Donor marked as "coming in" - prevent other hospitals from contacting them
3. Schedule their collection time

**Result: BLOOD SECURED IN 4-8 MINUTES** 🎯

## ⚡ KEY FEATURES AT A GLANCE

### Request Management
- **Create** blood requests in 2 minutes
- **View** all your hospital's active requests
- **Cancel** no-longer-needed requests (prevents wasted notifications)
- **Filter** by urgency or status

### Donor Matching
- **Ranked list** of 10 best donors (PAST-Match algorithm)
- **Compatibility score** shows why each donor was selected
- **Distance shown** in kilometers
- **Availability status** updated in real-time
- **Reliability score** from their donation history

### Response Management
- **Real-time tracking** updates every 5 seconds
- **Color-coded responses** (green/red/yellow) for quick scanning
- **Response count** shows how many accepted/declined/pending
- **Timeline** shows when each donor responded
- **Contact info** visible for calling donors directly

### Notifications
- **Notification history** of everything sent
- **Mark as read** to track what you've reviewed
- **Filter** by response type

## 🎯 PRO TIPS

### Tip 1: Use Critical Urgency
- Normal request = wait for responses
- Critical request = donors notified within seconds, top 5 fast-tracked
- **Use for:** Trauma, surgery, hemorrhage

### Tip 2: Watch the Top 3
- Don't scroll – focus on **Rank #1, #2, #3**
- These are the best matches
- 78% response rate from top 3 donors

### Tip 3: Parallel Notifications
- Can notify multiple donors at once
- Contact top 3 simultaneously
- Whoever responds first = you secure them
- Others automatically notified "request fulfilled"

### Tip 4: Check Donor Availability
- Green checkmark = Available right now
- Yellow exclamation = Recently notified (might be busy)
- Red X = Marked unavailable (skip)

### Tip 5: Use Notes for Data
- Add patient info in "notes": "Type & Cross, Baseline Hgb 7.5"
- Helps donors understand urgency
- Improves response rate
