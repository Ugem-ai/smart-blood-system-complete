# Admin Manual

## Admin Responsibilities

- Approve or reject hospital registrations.
- Monitor donor and request activity.
- Review analytics and system performance.
- Govern user accounts.

## Admin Dashboard

Key metrics:

- Total donations
- Total requests
- Active donors
- Pending hospitals
- Matching requests
- Completion rate

## Approval Workflow

1. Open pending hospitals list.
2. Verify submitted details.
3. Approve or reject hospital account.
4. Review activity logs for traceability.

## Analytics and Oversight

API and dashboard provide:

- Active donor count
- Requests today
- Success rate
- Average response time

## Audit and Compliance

Monitor activity logs for:

- admin approvals/rejections
- donor profile updates
- hospital access to request/matched donor records
- authenticated data access events

## Security Operations Checklist

- Keep token expiration active.
- Keep API throttling active.
- Review abnormal access patterns.
- Rotate secrets and credentials periodically.
